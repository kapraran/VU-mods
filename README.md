# VU Mods

A collection of simple mods for [Venice Unleashed](https://veniceunleashed.net/), the community-driven modding platform for Battlefield 3.

## Included Mods

| Mod | Description |
|-----|-------------|
| [balanced-reinforcements](balanced-reinforcements/) | Manage the number of reinforcements in Rush mode |
| [force-parachute](force-parachute/) | Forces the parachute to open at a configurable height |
| [fortnite-hit-effects](fortnite-hit-effects/) | Fortnite-style floating damage numbers when damaging players |
| [freecam-standalone](freecam-standalone/) | Standalone free camera for map navigation, extracted from [BF3RM/MapEditor](https://github.com/BF3RM/MapEditor) |
| [maped-last-save-loader](maped-last-save-loader/) | Load your last MapEditor save in a server |
| [minimal-vanilla-ui](minimal-vanilla-ui/) | Cleaner, less cluttered vanilla UI experience |
| [remove-flags-vehicles](remove-flags-vehicles/) | Example mod for removing flags and vehicles from maps |
| [soldier-spawn-plus](soldier-spawn-plus/) | Enhanced soldier spawn with console commands, fork of a mod from [EmulatorNexus/VEXT-Samples](https://github.com/EmulatorNexus/VEXT-Samples/tree/master/soldier-spawn) |

## Installation

Copy the desired mod folder(s) into your Venice Unleashed mods directory. Check individual mod READMEs for specific setup and configuration.
